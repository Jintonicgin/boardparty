# PRD (MVP 요구사항)

## 1. 목적
- 친구와 방을 만들어 티츄/라스베가스를 실시간 플레이
- 인원 부족 시 봇으로 채워 게임이 끝까지 진행
- 서버 권위 판정 + 재접속 + 이벤트 로그 저장

## 2. 플랫폼 (정식 지원)
- Web (Chrome / Edge / Safari 최신 2버전)
- Android (P0)
- iOS (P0)

※ Web / App은 동일한 게임 규칙과 서버를 사용하며,
  클라이언트 차이에 따른 기능 차별은 두지 않는다.

## 3. 범위(우선순위)

### P0 (필수)
- Web / Android / iOS 동일 기능 지원
- 게스트 로그인 + 닉네임
- 로비: 방 생성(코드), 입장, 나가기
- 방: 슬롯/Ready/Start, 봇 추가
- 실시간: Socket.IO 기반 상태 동기화
- 매치: 서버 권위 판정 + 턴 타이머 30초 + 재접속 90초
- 이벤트 로그 저장(match_events)
- Las Vegas 완성 (2~5인, 봇 포함)
- Tichu Full 룰 + 동점 타이브레이커(최대 2라운드) 완성(4인 팀전, 봇 포함)

### P1 (권장)
- iOS 안정화(TestFlight)
- 리플레이(이벤트 로그 재생)
- 텍스트 채팅 + 필터/뮤트
- 봇 난이도 4단계 확장

## 4. 성공 기준(DoD)
- 라스베가스/티츄가 **방 생성→플레이→종료**까지 99% 이상 정상
- 재접속(90초) 성공률 99%+
- illegal action 방어(클라 조작으로 결과 변경 불가)
- seed+events로 결과 재현 가능

## 5. 비범위 (Out of scope)
- 결제/상점
- 음성 채팅
- 정식 MMR / 랭킹
